# Skynet.CKEditor.FE

## Installation

#### Clone the repository

```bash
git clone git@github.com:Interfirst/Skynet.CKEditor.FE.git
cd Skynet.CKEditor.FE
yarn install
```

## Build

```bash
yarn build
```

## Start

```bash
yarn start
```

## Publish package
1. Build package locally (yarn run build)
2. Get npm auth token and add it to .npmrc
3. Run `npm publish`
